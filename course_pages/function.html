Functions in C++



    Functions are aset of statements that are grouped together to perform a specific taskfor repeated number of times.

    Types of Functions:

  • User-defined: These are the branch of functions that are defined by the user. It is basically customized bye the user to perform the task desired by the user and can be called n number of times from anywhere in the program.
    User defined functions have the following :
    • Function name
    • Return type
    • Function parameters
    • Function body

  • Built in: These are the functions already provided by the standard C++ libraries.These functions are defined in STL and header files too.


    • Advantages of C++ Functions:


  • Code Reusibility: The codes can be reused n number of times by just calling the function in which it is stored.

  • Code Debugging: The code can be debugged easily as it is divided into small small modules which makes it easier to understand where the actual error is.

  • Code Readibility: As the code is divided into modules it is easily readable and can be maintained and understood in a more easier way.
