Class Methods in C++



    C++ class methods are user-defined functions that are used inside an instance of the class. A dot notation . is used before method names to distinguish them from the regular functions.

    A class method can be defined in two ways:

  • Inside the class definition:
    • class Person
    {
    string name;
    public:
    // Defines the method
    void get_name()
    {
    return name;
    }
    }
    int main()
    {
    Person robert;
    // Calls the method
    robert.get_name();
    return 0;
    }
  • Outside the class definition
    • class Person
    {
    string name;
    public:
    void get_name();
    }
    // Defines the method
    void Person::get_name()
    {
    return name;
    }
    int main()
    {
    Person robert;
    // Calls the method
    robert.get_name();
    return 0;
    }
